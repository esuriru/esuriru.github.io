Their Last Ride is a 2D Windows game that was created in 2 weeks during my first semester of my diploma in NYP (Nanyang Polytechnic).

It was created in collaboration with my groupmates Ray, Tania, Paul and Jude during our Studio Project which we were randomly allocated as groupmates together. I was the lead programmer and project manager in the team.

Story

There is a fine line between life and death, and in this world, it's a train line. You are the Ticketmaster, and it is up to you to determine which passengers belong to the world of the living and the world of the dead.

Impacts

Programming

I programmed the renderer which utilized the free SDL2 library. I also created the asset import pipeline and created an API for my team to use to render sprites on the screen. This API was responsible for majority of the game and enabled my teammates to create some gameplay features. I also worked on gameplay features alongside the core of the framework like:

  • Gameplay Loop
  • JSON Level Loading
  • Dialogue System
  • UI Buttons/Picking
  • Font Loading

Teamwork

I had to work with teammates that were less capable than me at C++, not only that, but they had no knowledge of GitHub, a VCS (Version Control System) tool that is more or less mandatory when working with a team.

I learnt to be patient with my teammates and learnt that how to empathize with them and be realistic with my expectations from my teammates. This helped us to be able to finish our project in time for the deadline.
